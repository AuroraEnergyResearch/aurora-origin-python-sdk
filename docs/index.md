# Aurora Origin SDK

Python software development kit documentation for accessing Origin scenario metadata, workflows, and output data.

## Getting started

The SDK documentation is split into three parts:

- [Installation](installation) covers setup and authentication.
- [Common patterns](common-patterns) shows representative usage snippets.
- [API reference](api/index) is generated directly from the SDK's Python docstrings at build time.
